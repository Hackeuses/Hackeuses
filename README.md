# Hackeuses!

## The awesome landing page of our great project Hackeuses!


## Prerequisites:

### install Vuejs
> $ npm install -g vue-cli

### install GitFlow
> $ sudo apt-get install git-flow

## Usage:

### Git Flow
> create a local repo
> - $ cd {repo}
> - $ git flow init


### Clone the Remote Repo
> you should clone the repo
> - $ npm install


### Run:
> $ npm run dev
