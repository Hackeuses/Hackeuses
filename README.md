# Hackeuses!

## The awesome landing page of our great project Hackeuses!


## Prerequisites:

### install Vuejs
> $ npm install -g vue-cli

### install GitFlow
> $ sudo apt-get install git-flow

## Usage:

### Fork the Repo
> on your fork you should clone the repo
> $ npm install

### Git Flow
> $ git flow init

### Run:
> $ npm run dev
